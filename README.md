# 天翼云盘直链下载解析php版本

### 说明
因为个人需要在网上找到了很多脚本，但大部分都python和go语言的，或者油猴js脚本
最主要的是太复杂了，我只需要天翼云盘简单的直链解析，
所以简单研究了下，做了这个程序
适合把自己的云盘全部或者耽搁文件夹公开下载的朋友，或者说自己私用的下载站
